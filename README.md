# Burder-Builder 
- React and Redux (for frontend)
- Firebase (for backend)

## Scripts

| Script          | Description                                     |
| --------------- | ----------------------------------------------- |
| `npm install`   | Installs all dependencies                       |
| `npm start`     | Starts project in `development` environment     |
| `npm run build` | Builds the project for `production` environment |
